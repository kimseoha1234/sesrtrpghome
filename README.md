🎲 TRPG 종합 관리 매니저 (TRPG Manager)

TRPG(Tabletop Role Playing Game) 플레이어 및 마스터를 위한 개인용 종합 관리 대시보드입니다. 노션(Notion)과 서브컬처 개인홈의 감성을 담아 디자인되었습니다.

✨ 주요 기능

🗓️ 캘린더 대시보드: FullCalendar를 활용한 직관적인 일정 관리

📊 동적 데이터 분석: 달력에 일정을 등록하면 이번 달 남은 일정, 룰별 비율, GM/PL 비율이 자동으로 계산되어 우측 위젯에 표시됩니다.

📂 시나리오 아카이빙: 위시리스트, 룰북/시나리오집(소유), 주파 기록 등을 탭과 표 형태로 깔끔하게 분류하여 관리합니다.

🌙 라이트/다크 모드: 우측 하단의 달/해 모양 버튼을 통해 언제든 눈이 편안한 테마로 변경할 수 있습니다.

📱 반응형 디자인: PC와 모바일, 태블릿 어디서든 사이드바가 부드럽게 접히며 최적화된 화면을 제공합니다.

🛠️ 사용 기술 (Tech Stack)

HTML5 / JavaScript (ES6+)

Tailwind CSS (스타일링 및 다크모드 구현)

FullCalendar.js (달력 렌더링)

Phosphor Icons (아이콘 리소스)

🚀 사용 방법

별도의 백엔드 서버 없이 브라우저에서 즉시 실행 가능한 Single Page Application(SPA) 형태입니다.

index.html 파일을 다운로드합니다.

브라우저(Chrome, Edge 등)로 해당 파일을 열면 즉시 작동합니다.

데이터(일정, 캐릭터 등)를 수정하고 싶다면 index.html 내부의 events 배열 등 자바스크립트 데이터를 수정하면 됩니다.
